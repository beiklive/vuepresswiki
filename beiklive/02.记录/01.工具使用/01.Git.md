## 投机取巧
### add和commit合并成一条命令

`git commit -am "提交描述"`

### add  commit push合并

```bash
git config --global alias.acp '!f() { git add -A && git commit -m "$@" && git push; }; f'
```
使用：
```bash
git asp "xxxxx"
```
